Variables de Java

Mira, imagínate una variable en Java como una caja a la que le pones una etiqueta con un rotulador, por ejemplo "puntuacion", y dentro le metes un número. Básicamente son espacios en la memoria del ordenador que usamos para que nuestro programa no tenga memoria de pez y pueda recordar datos mientras funciona, actualizarlos (como cuando sumas puntos en un juego) y ahorrarnos tener que escribir el mismo valor cien veces por todo el código. La única regla pesada que tiene Java es que, antes de usar la caja, tienes que decirle de qué tamaño o tipo es; si es para números enteros le pides una caja tipo int, y si es para texto le pides un String, simplemente para asegurarse de que no intentas meter una bicicleta en una caja de zapatos.

Condicionales

"El condicional if es, en su esencia, el libre albedrío de tu programa. Antes de usarlo, tu código es como un tren atrapado en una sola vía: ejecuta instrucción tras instrucción, de forma lineal y ciega, sin importar los datos que reciba. Sin embargo, en el momento en que introduces un if, le otorgas a la máquina la capacidad de observar su entorno y reaccionar. Es la forma en la que el desarrollador le dice al ordenador: 'Detente, evalúa esta situación específica y, solo si la condición es verdadera, toma este camino alternativo'. Comprender el if no consiste simplemente en aprender a colocar paréntesis o llaves; consiste en desarrollar la lógica pura de la programación, aprendiendo a anticipar todos los escenarios posibles que puede sufrir tu programa y preparándose para ellos."
